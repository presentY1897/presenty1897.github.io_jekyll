# Function Component

함수로 React의 Component를 정의하는 방법이다. Hook을 이용하여 React의 life cycle과 여러 속성을 제어할 수 있다.

## 선언

```tsx
function FunctionComponent() {
  return <div>Function Component</div>;
}
```
